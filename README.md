# Paycheck Debt Planner — public site

Static pages served via GitHub Pages for the Paycheck Debt Planner iOS app:

- `index.html` — landing page
- `privacy.html` — Privacy Policy (App Store Privacy Policy URL)
- `support.html` — Support (App Store Support URL)

Source of truth lives in the app repo at `debt-app-v1/site/`; this repo is the
published copy. Pages: Settings → Pages → Source = `main` / root.
